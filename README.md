# Álbum Digital

Projeto desenvolvido com HTML, CSS e JavaScript com o objetivo de criar uma experiência interativa através de vídeos, música, animações e narrativa visual.

## Funcionalidades

- Galeria de vídeos interativa
- Reprodução de música ambiente
- Animações e transições personalizadas
- Efeito de carta interativa
- Design responsivo
- Modais para exibição de conteúdo multimídia

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi criado para explorar a criação de experiências digitais mais emocionais e imersivas, combinando elementos visuais, áudio e interatividade.
